# Задание 1. DevTools

## Авторизация пользователя

При логине пользователя на сайте вызывается запрос:

POST https://ilms.itmo.ru/api/accounts/public/auth/jwt/login

В теле запроса передаются учетные данные пользователя (логин и пароль).  
При корректных учетных данных сервер возвращает ответ со статусом **200 OK**,  
а также JWT-токены, которые используются для последующей авторизации пользователя
(access token и refresh token), а также информацию о роли пользователя.

---

## Получение данных пользователя

После успешного входа в систему и перехода в личный кабинет вызывается запрос:

GET https://ilms.itmo.ru/api/accounts/users/{id_пользователя}

В ответ сервер возвращает JSON-объект с информацией о текущем авторизованном пользователе,
включая идентификатор пользователя, адрес электронной почты, статус учетной записи
и роль в системе.

---

## Работа с DevTools

С помощью инструментов разработчика (DevTools) в браузере можно локально изменять
HTML-разметку страницы, а также просматривать сетевые запросы и ответы сервера
во вкладке **Network**.

В рамках задания продемонстрировано:
- изменение отображаемого текста страницы через вкладку **Elements**
- просмотр авторизованных сетевых запросов и ответов сервера

Подробная демонстрация представлена в видеофайле `dev_tools.mp4`.
